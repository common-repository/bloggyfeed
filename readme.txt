BloggyFeed shows your latest messages from Bloggy. 

== Description ==

With BloggyFeed you can show your latest messages from Bloggy.

= Features =

*	Simple
*	Easy to use
*	Widget support
*	Very Customizable
*	Swedish

= Usage =

If you are using Widgets, just activate it and do the customization there!

If you'r not a Widget-user you can use: `<?php bloggy("username"); ?>` to show your latest messages. For more info about the manual function, please visit [designskolan.net](http://www.designskolan.net/bloggy-plugin-for-wordpress/)

= Styling =

There are serveral CSS-classes that can be used. [Click here](http://www.designskolan.net/bloggy-plugin-for-wordpress/)
 for more info.


= News in BloggyFeed 1.0 =
 
*	Customize your link to the message
*	Choose between opening the link in a new window or in the current one.
*	Customize the timestamp
*	Show how long ago the message was post istead of when
*	Able to make links in messages clickable
 
== Installation ==

1. Drag the folder to the plugin-folder: `wp-content/plugins/`
1. Activate it!
